Untuk guru — cara menghubungkan ke Google Sheets/Drive (opsional)

Secara bawaan, jawaban siswa hanya tersimpan sementara di sesi browser masing-masing. Agar semua jawaban dari semua siswa otomatis terkumpul di satu Google Spreadsheet, dengan kolom yang sudah tersusun rapi sesuai urutan pertanyaan, ikuti langkah berikut:

Langkah 1 — Buat spreadsheet

  1. Buka sheets.google.com, buat spreadsheet kosong baru.
  2. Beri nama, misalnya "Rekap Pemetaan Bahasa Arab MAN Balikpapan".

Langkah 2 — Pasang skrip penerima data

  1. Di spreadsheet tadi, klik menu Extensions > Apps Script.
  2. Hapus semua kode bawaan di editor, lalu tempel kode berikut (kode ini otomatis membuat baris judul kolom sesuai urutan pertanyaan pada formulir, sehingga admin tinggal mengoreksi baris demi baris tanpa perlu menyusun ulang):
// Ganti kode admin di bawah ini agar sama dengan kode admin di file index.html
var ADMIN_KEY = 'guru123';

function doPost(e) {
  var ss = SpreadsheetApp.getActiveSpreadsheet();
  var d = JSON.parse(e.postData.contents);

  // aksi khusus: buka/tutup kuisioner (dipakai oleh panel admin)
  if (d.action === 'setStatus') {
    if (d.adminKey !== ADMIN_KEY) {
      return ContentService.createTextOutput(JSON.stringify({status:'error', message:'Kode admin salah'}))
        .setMimeType(ContentService.MimeType.JSON);
    }
    var configSheet = ss.getSheetByName('Config') || ss.insertSheet('Config');
    configSheet.getRange('A1').setValue(d.value === 'CLOSED' ? 'CLOSED' : 'OPEN');
    return ContentService.createTextOutput(JSON.stringify({status:'ok', value:d.value}))
      .setMimeType(ContentService.MimeType.JSON);
  }

  // default: simpan jawaban siswa
  var sheet = ss.getSheets()[0];
  var headers = [
    'Waktu Pengisian','Nama','NISN','Kelas','Tingkat','Program Peminatan','Jenis Kelamin',
    'Pemahaman - Membaca (Qira\'ah)','Pemahaman - Menulis (Kitabah)',
    'Pemahaman - Mendengar (Istima\')','Pemahaman - Berbicara (Kalam)',
    'Pemahaman - Kosakata (Mufradat)','Pemahaman - Tata Bahasa (Nahwu-Sharaf)',
    'Minat Umum (1-5)','Cara Belajar Favorit','Motivasi Utama',
    'Kesulitan yang Dirasakan','Kesulitan Lain','Saran'
  ];
  if (sheet.getLastRow() === 0) {
    sheet.appendRow(headers);
    sheet.getRange(1, 1, 1, headers.length).setFontWeight('bold');
    sheet.setFrozenRows(1);
  }
  sheet.appendRow([
    new Date(), d.nama, d.nisn, d.kelas, d.tingkat, d.jurusan, d.jenis_kelamin,
    d.p_membaca, d.p_menulis, d.p_mendengar, d.p_berbicara, d.p_kosakata, d.p_tatabahasa,
    d.minat_umum, d.metode, d.motivasi,
    d.kesulitan, d.kesulitan_lain, d.saran
  ]);
  return ContentService.createTextOutput(JSON.stringify({status:'ok'}))
    .setMimeType(ContentService.MimeType.JSON);
}

// endpoint ini yang dibaca form untuk tahu kuisioner sedang buka/tutup
function doGet(e) {
  var ss = SpreadsheetApp.getActiveSpreadsheet();
  var configSheet = ss.getSheetByName('Config');
  var status = 'OPEN';
  if (configSheet) {
    var val = configSheet.getRange('A1').getValue();
    if (String(val).toUpperCase() === 'CLOSED') status = 'CLOSED';
  }
  return ContentService.createTextOutput(JSON.stringify({status: status}))
    .setMimeType(ContentService.MimeType.JSON);
}
  1. Klik ikon disket/Save (atau Ctrl+S) untuk menyimpan skrip. Beri nama proyek bebas, misalnya "Penerima Kuisioner".

Langkah 3 — Deploy sebagai Web App

  1. Klik tombol biru Deploy di kanan atas > pilih New deployment.
  2. Klik ikon gerigi ⚙️ di samping "Select type" > pilih Web app.
  3. Pada "Execute as" pilih Me (akun Anda). Pada "Who has access" pilih Anyone.
  4. Klik Deploy. Google akan meminta otorisasi — klik Authorize access, pilih akun, lalu klik Advanced > Buka (nama proyek) (unsafe) jika muncul peringatan, lalu Allow. Ini normal karena skrip milik Anda sendiri.
  5. Setelah selesai, salin Web app URL yang muncul (formatnya https://script.google.com/macros/s/xxxxx/exec).

Langkah 4 — Hubungkan ke formulir ini (penting!)

  1. Tempel URL tadi ke kolom "URL Google Apps Script" di bawah ini, lalu klik Tes Koneksi untuk memastikan berhasil (akan muncul 1 baris data uji coba di spreadsheet — boleh dihapus setelah tes berhasil).
  2. Kolom di bawah ini hanya berlaku untuk browser Anda sendiri. Agar jawaban semua siswa otomatis masuk ke spreadsheet yang sama, URL tersebut harus ditempel langsung ke dalam file index.html sebelum dibagikan/di-upload ke Netlify — cari baris SCRIPT_URL: '' di bagian paling atas kode, lalu isi di antara tanda kutip: SCRIPT_URL: 'https://script.google.com/macros/s/xxxxx/exec'. Setelah itu, semua siswa yang membuka link otomatis terhubung tanpa perlu mengatur apa pun.

Catatan: jika suatu saat kode skrip diedit lagi, gunakan Deploy > Manage deployments > ikon pensil > New version > Deploy agar URL yang sama tetap berlaku dengan kode terbaru.

Langkah 5 — Menutup/membuka kuisioner

Setelah Langkah 4 selesai (URL sudah ditanam ke file), panel admin di bawah akan menampilkan tombol "Tutup Kuisioner" dan "Buka Kuisioner". Menutup kuisioner akan langsung menyembunyikan formulir dari semua siswa di semua perangkat sampai dibuka kembali — cocok dipakai saat periode pengisian sudah selesai.

Kolom ini otomatis terisi jika CONFIG.SCRIPT_URL sudah diisi di dalam file. Kosongkan/ubah di sini hanya untuk pengujian sementara di browser ini.

Unduh rekap jawaban (khusus admin/guru). Rekap ini hanya berisi jawaban yang masuk pada perangkat & sesi browser ini — bukan gabungan dari seluruh perangkat siswa. Untuk kumpulan data dari semua siswa, gunakan integrasi Google Sheets di atas lalu unduh langsung dari Google Drive.

Kode bawaan: guru123. Guru dapat mengganti kode ini di bagian atas file, cari ADMIN_CODE di dalam objek CONFIG.

Buka/Tutup Kuisioner. Menutup kuisioner akan menyembunyikan formulir dari semua siswa (di semua perangkat) sampai dibuka kembali. Butuh URL Google Apps Script di atas agar berlaku untuk semua orang, bukan hanya browser ini.
